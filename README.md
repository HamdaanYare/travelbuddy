## ✨ Features

### Core Features
- ✈️ **Flight Booking** - Search and book flights to destinations worldwide
- 🏨 **Hotel Reservations** - Find and book hotels with real-time availability
- 🎯 **Vacation Packages** - Pre-curated travel packages with discounts
- 👤 **User Authentication** - Sign up / Login system (local storage based)
- 💰 **Dynamic Pricing** - Real-time price calculations based on selections
- 📱 **Fully Responsive** - Works perfectly on desktop, tablet, and mobile

### User Experience
- 🔍 **Search Filters** - Filter flights and hotels by destination, date, and price
- 📅 **Date Picker** - Easy date selection for departures and check-ins
- ⭐ **Ratings System** - View hotel and destination ratings
- 💳 **Booking Confirmation** - Modal confirmation before final booking
- 🎨 **Modern UI** - Clean, gradient-based design with smooth animations
